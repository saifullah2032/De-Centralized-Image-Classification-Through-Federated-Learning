The FedAvg Algorithm

The Core Process
  • Local Training: Nodes train on local data slices.
  • Update Upload: Only weights are sent to the server.
  • Aggregation: Server averages updates via FedAvg.
  • Model Broadcast: Global model is sent back to all nodes.
